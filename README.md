# Lung Cancer Analytics Dashboard

## Overview
An end-to-end Lung Cancer Analytics project built using Databricks Medallion Architecture and Streamlit.

## Features
- Bronze, Silver, Gold Data Pipeline
- KPI Dashboard
- Risk Analytics
- Machine Learning Prediction
- Confidence Score
- Feature Importance
- Explainable AI (SHAP)
- Login System

## Tech Stack
- Databricks
- Python
- Pandas
- Streamlit
- Scikit-Learn
- Matplotlib
- SHAP

## Architecture
Bronze → Silver → Gold

## Dashboard
Interactive analytics dashboard with AI-powered risk prediction.
